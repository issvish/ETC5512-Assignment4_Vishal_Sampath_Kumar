Assignment 4- Quarto analysis of Melbourne parking congestion

## Author and Publishing date

This report was created by Vishal Sampath Kumar on the 5th June 2026 and published on 13th June 2026.

## Data Origin

The .csv files under data are based on City of Melbourne open datasets (please refer to the report for references).

The dataset used to derive "parking_processed.csv" is based on the On-street Parking Bay Sensors dataset.
The dataset used to derive supporting spatial summaries is based on the On-street Parking Bay Sensors GeoJSON export.
The supporting parking bay file is based on the On-street Parking Bays dataset.
